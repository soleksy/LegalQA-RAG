Polityka prywatności

Data ostatniej aktualizacji: 05.02.2024

Dziękujemy za korzystanie z naszej strony internetowej. Chcielibyśmy podkreślić, że nie zbieramy żadnych osobistych informacji od użytkowników na tej stronie. Nasza strona internetowa służy jedynie do prezentacji dostępnego API i nie gromadzi żadnych danych osobowych ani śledzi użytkowników.

Korzystanie z API

Nasza strona może udostępniać dostęp do różnych API lub usług innych firm. Korzystając z tych API lub usług, użytkownicy podlegają polityce prywatności tych firm. Prosimy o zapoznanie się z ich politykami prywatności przed użyciem tych usług.

Ciasteczka (Cookies)

Nasza strona może używać plików cookie wyłącznie w celu poprawy funkcjonalności strony. Pliki cookie nie są wykorzystywane do zbierania ani przechowywania danych osobowych.

Bezpieczeństwo

Dbamy o bezpieczeństwo naszej strony internetowej i stosujemy odpowiednie środki techniczne, aby chronić przed nieuprawnionym dostępem lub utratą danych.
